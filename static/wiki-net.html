Current Version: 1.5 Stable
5/30/2014
By Antonio Qiu

To my partners:

After v1.4, this program can not only run in the background, but also output the desired text file for Excel 2013. This means that the collection process has already become totally automated. After inputting it into Excel, we no longer have to do any search/replace.
Just Keep the text file updated, and refresh data in Excel whenever you want.
You can manually disable/hide the notifications in “Notification Area Icons” in Windows.

Try it out!

Change Log:

Instruction for 1.4
1. Run the program named “Path Sniffer.exe”, go to the second tab and do a simple setting, then launch it.
2. Open Excel. Select the desired starting cell, like the first one
     Find “import data from text file” in the Data tab.(Only for Office 2013; older Excel version might have it in “File”)
3. Go to the folder that stores that data file and select the file.
4. Original Data Type: Delimited – Next
     Delimiters: Other: copy “→” without””; – Next- Finish – OK
6. Just in case there are bugs, please check if there are rows with “No winners in the last game” or blank rows
7. Copy/Paste it into Google Docs.
